# BiasCorrGP
Repository containing code used in paper "Intuitively tuned elastic bias correction of atmospheric inversion using Gaussian process prior: Application to accidental radioactive emissions" by Antonie Brožová, Václav Šmídl, Ondřej Tichý and Nikolaos Evangeliou. 
https://www.sciencedirect.com/science/article/pii/S0304389426005017?via%3Dihub

It was executed with Python version 3.12.4.

The requirements are in requirements.txt package. 

Each dataset - ETEX-I, Chernobyl wildfires and Ruthenium release has a separate jupyter notebook, that demonstrates the use with the dataset.
